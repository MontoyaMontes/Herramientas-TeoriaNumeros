## Programa que efectua diferentes algoritmos referentes a teoría de números.

##  Uso

Se ejecuta desde consola

```console
user@user:~$ python3 herramientas.py
```

Hace uso de las librerias de python3:  
-  gcd
-  sys
- os
- math
